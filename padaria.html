Padaria Brasília

Projeto acadêmico com o intuito de criar um site para uma padaria. Neste caso o objetivo principal era por em pratica o aprendizado de Back-end, utilizando a linguagem Python e o framework Django. Foi desenvolvido para simular um E-commerce, onde o cliente/usuário pode se cadastrar, logo após, efetuar login e fazer seu pedido. Deste modo solucionando a demanda fictícia de diminuir as filas no local. Em uma versão futura, pretendo torná-lo funcional.

Status: Em desenvolvimento
